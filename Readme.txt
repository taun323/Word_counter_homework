This tool analyzes how many different words are used in a book.

The available books are the following:

Book 1, A Christmas Carol, by Charles Dickens
Book 2, Metamorphosis, by Franz Kafka
Book 3, The Adventures of Huckleberry Finn, by Mark Twain


The user is asked to enter a number from 1 to 3 to choose the book.
The result is the number of different words used in the selected book.
